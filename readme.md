# Card validator v.1.0.3

This library will check your credit card validation.

## Method

cardValidator(creditCardNumber);

###### Example:

You can use Node.js:

```
$node
> const check = require("card-card-validator")
> check(yourcreditcardnumberhere); // "true"/"false"
```
